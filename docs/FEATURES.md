# claw-chat 功能清单

claw-chat 是 OpenClaw 控制中心的 Android 客户端，用于和 OpenClaw 助手进行聊天交互，并管理文件和会话。

## 一、核心功能

### 1. 连接管理
- [ ] QR 扫码配对：扫描 OpenClaw Web UI 中的配对二维码，自动配置连接信息
- [ ] 手动输入配置：支持手动输入 Gateway URL 和 Token
- [ ] 连接测试：配对后测试连接是否正常
- [ ] 连接状态显示：实时显示当前连接状态（已连接/连接中/断开）
- [ ] 自动重连：网络断开后自动尝试重连
- [ ] 配置修改：在设置页面可以修改连接信息

### 2. 会话管理
- [ ] 会话列表：展示所有会话，显示会话名称、最近更新时间、未读消息计数
- [ ] 创建新会话：支持一键创建新的聊天会话
- [ ] 删除会话：点击删除按钮，二次确认后删除会话
- [ ] 重命名会话：支持修改会话名称
- [ ] 置顶会话：支持将会话置顶，置顶会话优先显示
- [ ] 排序：默认按置顶优先、更新时间倒序排列
- [ ] 搜索会话：支持按会话标题和消息内容搜索
- [ ] 归档/取消归档：支持将会话归档，归档后不在主列表显示
- [ ] 未读清零：手动清空未读计数
- [ ] 清空会话：清空会话中所有消息

### 3. 聊天交互
- [ ] 消息列表展示：区分用户消息和助手消息，气泡样式展示
- [ ] 输入框：支持多行输入、发送按钮、长按菜单
- [ ] 发送消息：点击发送按钮或回车发送消息
- [ ] 流式响应展示：助手回复支持逐字显示（打字机效果）
- [ ] 正在输入指示器：助手回复过程中显示 "正在输入..." 提示
- [ ] 一键滚动到底部：新消息不在可视区域时显示滚动按钮
- [ ] 富文本渲染：支持 Markdown 语法渲染，包括标题、列表、引用、链接
- [ ] 代码块高亮：代码块支持语法高亮展示
- [ ] LaTeX 公式：支持渲染 LaTeX 数学公式（可选）
- [ ] 消息操作：长按消息弹出操作菜单
  - [ ] 复制消息内容到剪贴板
  - [ ] 删除单条消息
  - [ ] 重新生成回复
  - [ ] 收藏消息（可选）
- [ ] 错误处理：发送失败显示错误提示，支持点击重试

### 4. 文件附件
- [ ] 附件选择：支持多种方式选择文件
  - [ ] 拍照：调用相机拍照后发送
  - [ ] 相册：从相册选择图片发送
  - [ ] 文件：从文件管理器选择任意文件
- [ ] 多选支持：支持同时选择多个文件发送
- [ ] 已选预览：发送前展示已选文件列表，支持移除已选文件
- [ ] 权限处理：相机、存储权限请求，权限被拒绝时引导用户去设置开启
- [ ] 文件上传：上传文件到 OpenClaw，展示上传进度
- [ ] 上传重试：上传失败支持点击重试
- [ ] 消息附件展示：在聊天气泡中展示附件缩略图/文件信息
  - [ ] 图片展示缩略图
  - [ ] 不同文件类型显示对应图标
- [ ] 文件预览：支持预览已接收的文件
  - [ ] 图片预览：全屏预览，支持手势缩放
  - [ ] 视频预览：内置播放器播放
  - [ ] 音频播放：内置播放器播放
  - [ ] PDF 预览：支持预览 PDF 文档
  - [ ] 不支持预览的文件显示占位卡片
- [ ] 文件下载：将接收到的文件下载到本地
- [ ] 保存到相册：图片/视频可以保存到系统相册
- [ ] 分享文件：支持将文件分享到其他应用
- [ ] 下载进度：展示下载进度

- [ ] 暗色主题
