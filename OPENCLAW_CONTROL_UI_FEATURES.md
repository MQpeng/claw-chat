## 侧边栏菜单顺序 (对齐 Control UI)

```
Overview  → 概览
Chat      → 聊天（会话列表）
Channels  → 频道
Sessions  → 会话完整管理（其实就是聊天，保留入口一致）
Cron Jobs → 计划任务
Skills    → 技能
Nodes     → 节点
Exec      → 执行审批
Config    → 配置
Logs      → 日志
Debug     → 调试
Update    → 更新
Settings  → 设置
```
